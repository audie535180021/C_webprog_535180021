1.1 Sejarah HTML

HTML adalah bahasa dasar untuk menampilkan halaman web pada web browser. HTML merupakan singkatan dari Hypertext Markup Language. Yang artinya adalah bahasa markup(penanda) berbasis text atau bisa disebut sebagai foramtiing language. HTML adalah bahasa pemrograman, HTML bukanlah bahasa pemrograman tetapi bahasa markup/formatting.

HTML dibuat oleh Tim Berners-Lee ketika masih bekerja untuk CERN dan dipopulerkan pertama kali oleh browser Mosaic. Selama awal tahun1990 HTML mengalami perkembangan yang sanyat pesat. Setiap pengembangan HTML pasti akan menambahkan kemampuan dan fasilitas yang lebih baik dari versi sebelumnya. Sebelum suatu HTML disahkan sebagai suatu dokumen HTML standar, ia harus disetujui dulu oleh W3C untukdievaluasi secara ketat. Setiap terjadi perkembangan suatu versi HTML, maka mau tak mau browser pun harus memperbaiki diri agar bisamendukung kode-kode HTML yang baru tersebut.
